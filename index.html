Bienvenido al Generador de Fichas

Esta herramienta está diseñada para ayudarte a crear personajes para el juego de rol **Call of Cthulhu 7e** de forma rápida y sencilla. Simplemente selecciona una de las opciones a continuación para comenzar.

¿Cómo Funciona?

Este generador te permite crear personajes de **Call of Cthulhu 7e** con las características básicas, como Fuerza (STR), Constitución (CON), Destreza (DEX), y más, usando tiradas de dados o permitiéndote ingresar los valores manualmente.

  • Generador de Personajes - Inicia la creación de tu personaje con las características básicas.
  • Reglas - Consulta las reglas de **Call of Cthulhu 7e** para conocer cómo se generan las características y otros aspectos del juego.
  • Contacto - Contáctanos si tienes dudas o sugerencias.

Características Clave del Generador

  • Generación automática de atributos mediante tiradas de dados.
  • Posibilidad de introducir valores manualmente para las características.
  • Visualización en tiempo real de los datos en formato JSON.
